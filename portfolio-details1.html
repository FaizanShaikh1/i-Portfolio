Project Details

Power BI Project: Visualizing OTT Platforms Data

Objective:

Analyzing data from various OTT platforms like Netflix, Hotstar, and Amazon Prime to provide comprehensive insights into movies, shows, and series.

Project Learnings:

Created an interactive dashboard for real-time tracking and user-driven analysis. Established data connections, performed table joins, and conducted data manipulations to enable parameter-driven visualizations. Utilized a wide range of customized visualizations including bar charts, pie charts, donut charts, clustered bar charts, scatter charts, line charts, area charts, maps, slicers, and more.

Key Project Points:

Data Collection, Data Cleaning and Wrangling, Data Modeling, Data Visualization, Dashboard Design, Sharing Insights.

Dashboard Features:

Total Shows, Movies, Series, IMDB Score, Genre, List of Shows, Rating, Durations, Awards, Year of Release, Director name, About Shows.
