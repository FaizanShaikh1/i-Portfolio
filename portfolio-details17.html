This interactive visualization aims to provide valuable insights and analysis about the pandemic situation in our country. This project was created using Tableau Desktop.

Dashboard Components:

  • Total Deaths By State: This section provides insights into the total number of COVID-19-related deaths in each state.

  • First & Second Dose Administered: A visual representation of the number of people who have received their first and second doses of the vaccine.

  • ICMR Testing Lab In Each State: Information on the presence of ICMR testing labs in various states.

  • Doses Administered By Vaccine: Analysis of vaccine doses administered categorized by the type of vaccine.

  • State Wise Testing Details: Comprehensive testing details for each state.

  • Covid Cases By State: Visualization of COVID-19 cases by state.

Key Features of the COVID-19 Dashboard:

  • Vaccines: Age Group Analysis

  • Confirmed Cases: State-wise Distribution

  • Total Deaths: State-wise Statistics
